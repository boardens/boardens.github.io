Indexing pages

You can configure sections organisation and pages indexing by editing config/index.json.

{
  "index":[
    {
      "section":"Lorem ipsum",
      "items":[
        {
          "title":"Hello world",
          "src":"index.md"
        },
        {
          "title":"Components",
          "src":"components.md"
        }
      ]
    }
  ]
}


Pages

Please note that the first page should be indexed as index.md.

{
  "title":"Lorem ipsum",
  "src":"lorem-ipsum.md"
}
Argument Description
title Page title
src Source markdown file


Sections

{
  "section":"Lorem ipsum",
  "items":[]
}
Argument Description
section Section title
items List of pages in this section
